В окружении Python 3.11 +

pip install -r requirements.txt

python3 adduser.py

Внимание: Генерация закрытого и открытого ключей

Шаг 1: Откройте этот сайт в новой вкладке и нажмите кнопку «Сгенерировать новые ключи», чтобы создать закрытый и открытый ключи.

http://travistidwell.com/jsencrypt/demo/

Шаг 2: Скопируйте сгенерированный закрытый ключ и откройте этот сайт кодировки base64, чтобы преобразовать его в Base64.

https://www.base64encode.org/

Нам нужно закодировать ключи в base64, чтобы избежать ненужных предупреждений в терминале.

Шаг 3: Скопируйте ключ в кодировке base64 и добавьте его в файл .env как JWT_PRIVATE_KEY .

Шаг 4: Вернитесь на сайт генерации открытых/закрытых ключей и скопируйте соответствующий открытый ключ.

Шаг 5: Вернитесь на сайт кодировки base64, чтобы преобразовать открытый ключ в base64 и добавить его в файл .env как JWT_PUBLIC_KEY


Запустить

uvicorn --reload main:app

Внимание, при запуске автоматически очищается база данных, добавляется модератор и ключи
